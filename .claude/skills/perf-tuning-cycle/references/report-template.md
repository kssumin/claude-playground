## 판정 기준

| 판정 | 기준 | 다음 액션 |
|------|------|---------|
| ✅ 개선 | p95 개선 OR 내부 지표 개선 | 변경 유지 + 커밋 |
| ⚠️ 부분 | 일부 지표만 개선 | 근본 원인 추가 분석 |
| ➖ 무효 | p95 ±5% 이내 | 사용자 판단 (안정성 향상이면 유지) |
| ❌ 악화 | p95 악화 OR 에러율 증가 | 즉시 롤백 |
